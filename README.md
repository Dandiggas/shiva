# Shiva

![shiva](logo.png)

Shiva library: Implementation in Rust of a parser and generator for documents of any type

Current status of Shiva library:

| Document type | Parse | Generate |
|---------------|-------|----------|
| Plain text    | +     | +        |
| Markdown      | +     | +        |
| HTML          | +     | +        |
| PDF           | -     | -        |
| JSON          | -     | -        |
| XML           | -     | -        |
| DOC           | -     | -        |
| XLS           | -     | -        |
